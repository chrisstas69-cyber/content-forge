# Introduction

This document is your complete master guide to ContentForge. It contains three parts:

Part 1: 23 video scripts with narration, visual cues, and pro tips. Use these to record tutorial videos or generate AI videos.

Part 2: A field-by-field guide explaining every input, dropdown, toggle, and button across all 20 tabs. Use this as your reference manual.

Part 3: A recording checklist to ensure professional-quality tutorials every time.

**How to use the video scripts:**
1. Each video has NARRATION (what to say), VISUAL (what to show), and PRO TIP sections.

2. For screen recording: Follow the VISUAL cues while reading the NARRATION.

3. For AI video generation (Higgsfield, etc.): Use NARRATION as script input, describe VISUAL as scene directions.

4. Videos are self-contained. Record in any order or combine into longer compilations.

**Recording tips:**
- Record in 1080p or 4K so viewers can read UI text when paused
- Use a clean browser profile (no bookmarks bar, no extensions)
- Zoom into UI elements using screen magnifier
- Add captions/subtitles for accessibility
- Keep a consistent intro/outro across all videos

- CapCut (FREE) - Best free editor, desktop + mobile

- iMovie (FREE, Mac) - Simple, good for beginners
- Adobe Premiere Pro ($22/mo) - Industry standard
